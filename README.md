# Day 1: Study Status Manager 🚀

a simple script on Python to organize the study process created on my first day on solo study.

## 🥁 How does it work?
- It collects info about the subject's name, progress percent and the next task via keyboard
- Save all info automatically on a local `.txt` file.
- Keeps a log (it doesn't delete what has been written before).

## 🔥 Today's learning:
- File manipulation by using `with open()` and `append ("a")`.
- Use of  **f-strings** to organize variables inside of texts.
- Control of escape character's flow (`\n`) to organize lines.
- Basic code documentation in english.
