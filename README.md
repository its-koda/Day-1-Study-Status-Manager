# Study Status Manager | Gerenciador de Status de Estudo 🚀

<details open>
  <summary>🌐 <b>English Version</b></summary>
  <br />

  A simple Python script to organize the study process, created on my first day of solo study.

  ## 🥁 How does it work?
  * It collects info about the subject's name, progress percentage, and the next task via keyboard.
  * Saves all info automatically in a local `.txt` file.
  * Keeps a history log (it doesn't overwrite previous records).

  ## 🔥 Today's learning:
  * File manipulation using `with open()` and append mode (`"a"`).
  * Use of **f-strings** to format and organize variables inside texts.
  * Control of escape characters (`\n`) to handle line breaks and text organization.
  * Basic code documentation in English.

  ---
  *A simple exercise with a main focus on Logic and organization.*

  Um script simples em Python para organizar o processo de estudo, criado no meu primeiro dia de estudos solo.

  ## 🥁 Como funciona?
  * Coleta informações sobre o nome da matéria, porcentagem de progresso e a próxima tarefa via teclado.
  * Salva todas as informações automaticamente em um arquivo `.txt` local.
  * Mantém um histórico de logs (não apaga o que foi escrito anteriormente).

  ## 🔥 Aprendizados de hoje:
  * Manipulação de arquivos utilizando `with open()` e o modo de adição (`"a"`).
  * Uso de **f-strings** para formatar e organizar variáveis dentro de textos.
  * Controle de caracteres de escape (`\n`) para organizar quebras de linha.
  * Documentação básica de código em inglês.

  *Um exercício simples com foco principal em Lógica e organização.*
